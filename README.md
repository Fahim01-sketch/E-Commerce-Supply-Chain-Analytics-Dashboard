# E-Commerce & Supply Chain Analytics Dashboard

## Overview
This project analyzes 10,000+ e-commerce transactions to evaluate sales performance, profitability, customer behavior, and supply chain efficiency.

## Key Insights
• 37.4% of revenue was impacted by returns and cancellations.
• Discounts of 40–50% increased sales volume but resulted in negative profit margins.
• Overnight shipping cost over 12× more than economy shipping while providing similar delivery times.
• Repeat customers generated significantly higher revenue than one-time buyers.

## Tools Used
• Power BI
• Power Query
• DAX
• Excel

## Business Value
The analysis identifies opportunities to improve profitability, optimize shipping costs, and strengthen customer retention strategies.
